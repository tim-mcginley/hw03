ReadMe for code/

This directory contains code I have written to do most of the analysis contained in the final report.  

functions/ contains regression functions for the computation of various summary stats.

scripts/ contains a few .R scripts that compute tables and figures.
eda-script.R takes the data and computes summary statistics of it
regression-script.R computes the linear regression model and a summary of it

tests/ contains tests to ensure that functions are acting as normal.
